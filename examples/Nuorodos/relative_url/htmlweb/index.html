Pagrindinis

2017-05-16

Šiandien pradedame HTML, CSS ir JavaScript paskaitų ciklą. Grupė yra gerai susipažinusi su programavimo pagrindais. 4 savaites iki šio paskaitų ciklo intensyviai mokėsi C#, JAVA, SQL. Išmoko pagrindinius programavimo principus.

Šiuo metu mokomės HTML, vėliau mokysimės CSS ir JavaScript. Skirtingai nei JavaScript HTML ir CSS nėra programavimo kalbos. HTML yra žymėjimo kalba, kai tuo tarpu CSS yra žymėjimo kalbos HTML aprašomoji kalba.
